# 🎯 人才画像师

> AI招聘决策系统 · Agent 1 / 3

从核心使命出发，经过任务拆解，倒推出结构化人才标准——帮你在招聘开始之前，把"找什么样的人"这个问题彻底想清楚。

---

## 快速上手

**第一步：打开 Bot**
前往扣子平台，搜索「人才画像师」或「HR小酒馆」，找到对应 Bot。

**第二步：准备材料（越详细，输出质量越高）**
- 岗位 JD（有的话直接贴全文）
- 公司官网链接
- 口头描述也可以，但建议补充：公司规模、行业、这个岗位汇报给谁、预算范围

**第三步：完成 6 步对话**

| 步骤 | 内容 |
|------|------|
| 1 | 上传 JD 和官网，Bot 自动分析 |
| 2 | 确认核心使命方向（3 选 1，可多选加权） |
| 3 | 确认任务清单（8-12 项，可增删调优先级） |
| 4 | 分层确认能力要求（底层能力 → 应用能力 → 项目经历 → 知识资源） |
| 5 | 选择 JD 风格（专业严谨 / 轻松亲和） |
| 6 | 生成完整报告 |

全程约 10-15 分钟，输出一份包含 10 个板块的完整人才画像报告。

**⚠️ 使用前必看**

使命选错，全盘偏航。第 2 步的三个使命方向代表三种完全不同的人，不要随手选——花 30 秒认真想清楚这个岗位最核心的任务是什么，再选。

---

## 报告包含哪些内容？

1. 岗位基本信息 + 核心使命
2. 完整 JD（可直接发布）
3. 核心任务清单（含 AI 影响标注）
4. 四层能力画像 + 推导连线图 ⭐
5. 薪酬建议（P25 / P50 / P75 三分位）
6. 精确降级方案 + 候选人拓展地图 ⭐
7. 场景适配验证（含面试追问话术）
8. AI 影响深度提示
9. 推荐招聘渠道组合
10. 关键决策总结

> ⭐ 标注的两个板块是使用者反馈最有价值的部分——建议重点阅读。

## 当前版本局限

坦诚说清楚，避免你用了之后产生不必要的落差。

**平台限制**
运行在扣子平台，底层模型为豆包。推理深度和稳定性不如 Claude / GPT-4，偶尔会出现：
- 输出质量波动（输入越详细，质量越稳定）
- 内部思考过程"泄漏"到输出里（刷新重跑一次即可）

**无跨对话记忆**
每次使用都从零开始，不记得你上次分析的岗位或公司背景。同一个公司招多个岗位，需要每次重新输入公司信息。

**无可视化图表**
报告里的雷达图数据已生成，但平台不支持直接渲染图表——需要自己把数值复制到 Excel 画图。

**不替代你的判断**
工具能帮你推导，但"使命选哪个""红线怎么判""场景风险够不够大"——这些判断工具给不了。工具越强，用错的代价越大。
