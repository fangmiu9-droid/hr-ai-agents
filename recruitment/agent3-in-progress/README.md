## 它将解决什么问题？

人才画像师帮你想清楚要什么人，雇主镜像师帮你把对的人吸引来——但候选人到了面试桌对面，怎么评估、怎么谈、怎么让他真的留下来？

这是 Agent 3 要解决的。

**核心功能规划：**
- 差异化面试考核方案（基于人才画像师的输出定制）
- Offer 谈判策略（薪酬 + 非薪酬要素的组合方案）
- 90 天融入风险预警（提前识别可能的水土不服信号）
