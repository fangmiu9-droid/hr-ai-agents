# 🏢 雇主镜像师

> AI招聘决策系统 · Agent 2 / 3

帮你从候选人的视角照一面镜子——你以为自己传递的是 A，候选人接收到的可能是 B。七维度扫描品牌一致性，诊断你的 JD、口碑和面试体验是否说的是同一件事。

---

## 快速上手

**第一步：打开 Bot**
前往扣子平台，搜索「雇主镜像师」或「HR小酒馆」，找到对应 Bot。

**第二步：准备材料（这个 Agent 对输入质量极度敏感）**

| 材料 | 重要程度 |
|------|---------|
| 岗位 JD 全文 | ⭐⭐⭐ 必须 |
| 公司官网链接 | ⭐⭐⭐ 必须 |
| 公司文化描述（你认为的） | ⭐⭐ 建议提供 |
| 面试官的开场白 / 常用话术 | ⭐⭐ 建议提供 |
| 公司硬件信息（规模、融资、办公环境等） | ⭐ 有的话提供 |

> 垃圾进，垃圾出。你花 2 分钟贴一个 JD，拿到的是通用废话。你花 20 分钟把材料备齐，拿到的是精确到每一句用词的品牌诊断。

**第三步：完成 7 步对话**

| 步骤 | 内容 |
|------|------|
| 1 | 上传 JD + 官网，选择公司文化风格原型 |
| 2 | 品牌全景扫描（文化信号提取 + 未被展示的卖点） |
| 3 | 口碑层扫描（联网检索脉脉 / 知乎 / 小红书，做一致性核查） |
| 4 | 面试体验层（输入面试官话术，分析文化信号是否一致） |
| 5 | 输出候选人视角的品牌画像 + 一致度评分 |
| 6 | **暂停确认**：你选择品牌定位方向（AI 不替你做战略取舍） |
| 7 | 输出三套改进方案 + 面试行为指南 + JD 重写建议 |

全程约 15-20 分钟。

**⚠️ 第 6 步是关键**

Bot 会暂停，让你确认品牌定位方向。这一步不要跳过——不同的定位选择，对应完全不同的改进方案。你选了"人性化温馨"和选了"高压高回报"，后面的建议是两套完全不同的东西。

---

## 报告包含哪些内容？

1. 候选人视角的品牌画像（"你在候选人眼中长这样"）
2. 品牌一致度评分（文案 / 口碑 / 面试三层对照）
3. 被浪费的卖点清单（有优势但没展示出来的）
4. 品牌三功能评分（吸引力 / 筛选力 / 预期管理力）
5. 三套改进方案（轻量版 / 标准版 / 深度版）
6. JD 重写建议（精确到每一处用词）
7. 面试行为指南（从你的定位推导，不是通用模板）

## 当前版本局限

**平台限制**
同 Agent 1，运行在扣子平台，底层模型为豆包。输入材料越完整，输出越稳定。

**口碑搜索覆盖不完整**
联网搜索能覆盖脉脉、知乎、小红书的公开内容，但：
- 部分平台需要登录才能看到的内容无法抓取
- 小公司口碑数据少，这一层诊断参考价值有限
- 搜索结果有时效性，不代表最新口碑状态

**无持续追踪能力**
理想版本应该能记录你每次改进前后的一致度变化，对比提升了多少。当前版本没有记忆，每次都是独立诊断。如果你想追踪改进效果，建议自己截图保存每次的评分结果。

**诊断不等于决策**
工具告诉你"品牌一致度 60%、这三处有矛盾"——改不改、怎么改、说服不说服面试官落地，是你的判断和执行。
