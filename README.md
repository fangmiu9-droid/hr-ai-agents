## 为什么做这个？

当前大多数AI应用，聚焦在**个人效率提升**——写作、搜索、总结。  
但组织层面的应用，严重匮乏。

这个问题在HR领域尤为突出：

- 现有HR软件对AI的集成浮于表面，缺乏真正的深度应用
- 大量组织管理的底层理念，还停留在前AI时代，亟需重构
- 与此同时，组织不可能一夜切换——**人与AI共存，是相当长一段时间内的现实**

我在阿里巴巴、万达、远洋地产等企业做过数千次人才评估，深知组织决策的复杂性。  
过去几年，我持续研究AI在组织运作中的实际落地场景，核心判断只有一个：

**所有过往的组织管理理念，都需要在AI时代重新审视。**

这个仓库，是我把研究成果转化为可用工具的公开记录。  
也是一个邀请——希望更多HR同仁真正学会利用AI的杠杆，抓住这一轮技术更新的红利，而不是在观望中错过它。
